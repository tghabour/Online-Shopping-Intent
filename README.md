# Online Shopping Intent

## Objective:

Predict online purchasing outcomes based on browsing behavior and other features.

## Featured Techniques:

- Supervised Learning
- Classification
  - Logistic Regression 
  - k-NN 
  - Naive Bayes
  - Decision Trees 
  - Random Forests
- Feature Engineering 
- Data Visualization
- Ensembling
